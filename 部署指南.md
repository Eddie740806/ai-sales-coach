# 🚀 直接部署到云平台 - 5分钟上线

## ✅ 平台已准备好部署！

**没有技术障碍**，平台可以直接部署到云服务。以下是三种最简单的方法：

---

## 🎯 方法1：Railway（推荐，最简单）

### 步骤：

1. **访问 Railway**
   - 打开：https://railway.app
   - 用 GitHub 账号登录（免费）

2. **新建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的仓库（如果没有，先推送到 GitHub）

3. **自动部署**
   - Railway 会自动检测到 `railway.json`
   - 自动安装依赖和启动
   - **完成！** 会给你一个 URL，例如：`https://your-app.railway.app`

4. **配置环境变量**（可选）
   - 在 Railway 项目设置中添加：
     ```
     OPENAI_API_KEY=your_key_here
     OPENAI_MODEL=gpt-3.5-turbo
     ```

**优点**：
- ✅ 免费额度充足
- ✅ 自动 HTTPS
- ✅ 自动部署
- ✅ 无需配置

---

## 🎯 方法2：Render（免费，简单）

### 步骤：

1. **访问 Render**
   - 打开：https://render.com
   - 注册账号（免费）

2. **新建 Web Service**
   - 连接 GitHub 仓库
   - 选择你的项目

3. **配置**
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `uvicorn server.main:app --host 0.0.0.0 --port $PORT`
   - Environment: `Python 3`

4. **部署**
   - 点击 "Create Web Service"
   - 等待部署完成
   - 获得 URL：`https://your-app.onrender.com`

**优点**：
- ✅ 免费套餐可用
- ✅ 自动 HTTPS
- ✅ 简单配置

---

## 🎯 方法3：Fly.io（免费，快速）

### 步骤：

1. **安装 Fly CLI**
   ```bash
   curl -L https://fly.io/install.sh | sh
   ```

2. **登录**
   ```bash
   fly auth login
   ```

3. **初始化并部署**
   ```bash
   fly launch
   fly deploy
   ```

4. **获得 URL**
   - 部署完成后会显示 URL
   - 例如：`https://your-app.fly.dev`

## 📋 部署前准备

### 1. 推送到 GitHub（如果还没有）

```bash
# 初始化 git（如果还没有）
git init

# 添加文件
git add .

# 提交
git commit -m "AI Sales Coach Platform"

# 推送到 GitHub
git remote add origin https://github.com/你的用户名/ai-sales-coach.git
git push -u origin main
```

### 2. 创建 .gitignore（如果还没有）

确保 `.gitignore` 包含：
```
venv/
__pycache__/
*.pyc
.env
*.db
chroma_db/
.DS_Store
```
