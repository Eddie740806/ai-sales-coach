# 前端使用指南

## 🎨 UI界面已创建完成！

前端界面已经创建在 `frontend/` 目录下，包含：

- `index.html` - 主页面
- `css/style.css` - 样式文件
- `js/app.js` - JavaScript逻辑

## 🚀 访问方式

### 方式1：直接打开HTML文件
在浏览器中直接打开 `frontend/index.html` 文件

### 方式2：通过FastAPI服务器（推荐）
服务器已经配置为提供静态文件服务，访问：
- `http://localhost:8000/` - 前端界面

## 📱 功能页面

### 1. 对话教练
- 业务员与AI教练实时对话
- 查看AI建议和推荐话术
- 支持设置客户类型

### 2. 知识库管理
- 培训师添加培训内容
- 搜索和浏览现有内容
- 支持多种内容类型

### 3. 话术生成器
- AI辅助生成销售话术
- 自动生成多个变体版本
- 支持场景和客户类型定制

### 4. 数据统计
- 查看平台整体数据
- 对话数、活跃用户数统计
- 表现最佳的话术排行

### 5. 学习洞察
- 查看业务员的个性化建议
- 基于历史对话的分析
- 推荐相关培训内容

## 🎯 快速开始

1. 确保后端服务器正在运行（`uvicorn server.main:app --reload`）
2. 打开浏览器访问 `http://localhost:8000/`
3. 开始使用！

## 💡 使用提示

- **对话功能**：在"对话教练"页面输入问题，AI会立即回复
- **知识库**：培训师可以在"知识库"页面添加新的培训内容
- **话术生成**：在"话术生成"页面填写场景和要求，AI会生成专业话术
- **数据统计**：实时查看平台使用情况
- **学习洞察**：输入业务员ID查看个性化训练建议
