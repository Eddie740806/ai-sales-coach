# 🎉 分享给朋友 - 快速指南

## ✅ 平台已准备就绪！

你的 AI 销售教练平台已经可以分享给朋友了！

## 📱 访问地址

### 同一 WiFi 网络访问：
```
http://192.168.50.182:8000
```

**重要**：确保你和朋友连接在**同一个 WiFi 网络**上。

## 🚀 使用步骤

### 给朋友的操作指南：

1. **打开浏览器**
   - 在地址栏输入：`http://192.168.50.182:8000`
   - 按回车访问

2. **注册账号**
   - 点击页面上的登录按钮
   - 切换到"注册"标签
   - 填写：
     - 用户名（至少3个字符）
     - 密码（至少6个字符）
     - 邮箱（可选）
     - 姓名（可选）
     - 角色：选择"業務員"
   - 点击"註冊"

3. **开始使用**
   - 注册成功后会自动登录
   - 可以测试以下功能：
     - 💬 **對話教練** - 与AI教练对话
     - 📚 **知識庫** - 搜索培训内容
     - 📝 **話術生成** - 生成销售话术
     - 📊 **數據統計** - 查看数据仪表板
     - 🎯 **學習洞察** - 查看个性化建议

## ⚠️ 注意事项

1. **服务器必须保持运行**
   - 如果关闭服务器，朋友无法访问
   - 建议保持电脑开机

2. **防火墙设置**
   - 如果朋友无法访问，检查防火墙设置
   - macOS：系统偏好设置 → 安全性与隐私 → 防火墙

3. **网络要求**
   - 必须在同一局域网（同一WiFi）
   - 如果不在同一网络，需要使用 ngrok（见下方）

## 🌐 如果不在同一网络

### 使用 ngrok（内网穿透）：

1. **安装 ngrok**
   ```bash
   brew install ngrok
   # 或访问 https://ngrok.com/download
   ```

2. **启动隧道**
   ```bash
   ngrok http 8000
   ```

3. **分享 ngrok 提供的 URL**
   ```
   例如：https://abc123.ngrok.io
   ```
   朋友可以通过这个 URL 访问，即使不在同一网络！
