## 🚀 立即开始

### 1. 访问API文档
打开浏览器访问：`http://localhost:8000/docs`

### 2. 测试对话功能
在API文档页面，找到 `/api/chat` 端点，点击 "Try it out"，输入：
```json
{
  "message": "客户说价格太贵，我该怎么回应？",
  "sales_id": "sales_001",
  "customer_type": "中小企业"
}
```

### 3. 查看示例数据
系统已自动加载5个示例培训内容：
- 客户需求挖掘技巧
- 价格异议处理话术
- 建立信任关系的最佳实践
- 成交促成技巧
- 中小企业客户沟通策略

### 4. 生成话术
测试 `/api/scripts/generate` 端点，生成定制化销售话术。

## 🔑 配置OpenAI（可选）

如果你有OpenAI API Key，可以：

1. 创建 `.env` 文件：
```
OPENAI_API_KEY=sk-your-key-here
OPENAI_MODEL=gpt-3.5-turbo
```

2. 重启服务器，即可使用真实GPT-3.5 Turbo模型。

**注意：** 即使没有API Key，系统也会使用智能回退模式，提供专业的建议。
