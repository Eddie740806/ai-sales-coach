## 📋 部署步骤

### 步骤 1：推送到 GitHub（如果还没有）

```bash
# 1. 初始化 git（如果还没有）
git init

# 2. 添加所有文件
git add .

# 3. 提交
git commit -m "AI Sales Coach Platform - Ready for deployment"

# 4. 在 GitHub 创建新仓库（网页操作）
#    访问：https://github.com/new
#    仓库名：ai-sales-coach
#    选择 Public 或 Private
#    不要初始化 README、.gitignore、license

# 5. 连接并推送
git remote add origin https://github.com/你的用户名/ai-sales-coach.git
git branch -M main
git push -u origin main
```

### 步骤 2：部署到 Railway

1. **访问 Railway**
   - 打开：https://railway.app
   - 点击 "Start a New Project"

2. **登录**
   - 选择 "Login with GitHub"
   - 授权 Railway 访问你的 GitHub

3. **部署项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的 `ai-sales-coach` 仓库
   - Railway 会自动检测并开始部署

4. **等待部署**
   - 大约 2-5 分钟
   - 会显示部署日志

5. **获得 URL**
   - 部署完成后，点击项目
   - 点击 "Settings" → "Domains"
   - 会看到类似：`https://your-app.railway.app`
   - **这就是你的公开 URL！**

### 步骤 3：配置环境变量（可选）

如果需要 OpenAI API：

1. 在 Railway 项目页面
2. 点击 "Variables"
3. 添加：
   ```
   OPENAI_API_KEY = your_key_here
   OPENAI_MODEL = gpt-3.5-turbo
   ```
